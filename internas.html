UNIDADES DE LECTURA

Sigla del inglés compact disc read-only memory ( 'disco compacto de solo lectura' ) Dispositivo que se utiliza para acceder, leer, copiar y hacer copias de seguridad de datos de una amplia variedad de tarjetas de memoria, incluyendo SD (Secure Digital), CF (CompactFlash), MMC (MultiMediaCardC) y muchas otras.

DISCOS DE ALMACENAMIENTO

Disco duro o HDD (Hard Drive Disk)

Están compuestos de piezas mecánicas, de ahí que a veces se le llame discos duros mecánicos, y utilizan el magnetismo para grabar tus datos y archivos.

 Conocida como unidad de estado sólido o disco de estado sólido, una SSD (por la abreviación en inglés de Solid State Drive) es una opción de almacenamiento de datos que funciona sin partes móviles.
